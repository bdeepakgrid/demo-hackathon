# 🔍 SynthScan — AI Image Forensics

> Detect AI-generated images in seconds using a Vision Transformer model with Grad-CAM explainability.

🌐 **Live Demo:** https://huggingface.co/spaces/Dkbn1324/SynthScanner 

---

## 🧠 What It Does

| Feature | Description |
|---|---|
| 🖼 Image Model | Upload a single image for instant forensic analysis + Grad-CAM heatmap |
| 📦 Batch & Monitor | Analyze up to 20 images at once, or watch a folder in real-time |
| 🌐 Web Scanner | Paste any URL — scans all images on the page automatically |
| 🎥 Video Analysis | Upload a video; analyzed frame-by-frame for deepfake detection |
| 💬 AI Chat | Ask questions about your results via the built-in assistant |
| ⭐ Reviews | Leave a star rating and feedback directly on the site |

---

## 🛠 Tech Stack

- **Frontend:** Vanilla HTML / CSS / JavaScript
- **Backend:** Python Flask + Gunicorn
- **AI/ML:** PyTorch, HuggingFace Transformers, Grad-CAM
- **Video:** FFmpeg
- **LLM Chat:** Groq API
- **Deployment:** Docker on HuggingFace Spaces
